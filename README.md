# 🧠 Resume Screening & Ranking System (Task 3)

## 📌 Overview
This project builds an AI-based resume screening system that automatically classifies resumes into job categories and ranks candidates based on relevance.

It simulates real-world Applicant Tracking Systems (ATS) used in HR technology.

---

## 🎯 Objective
- Classify resumes into job roles
- Extract meaningful features from resume text
- Rank candidates based on job relevance
- Automate resume screening process

---

## 🛠️ Tools & Libraries
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF Vectorization)
- Logistic Regression

---

## ⚙️ Workflow
1. Load and clean resume dataset
2. Convert text into numerical features (TF-IDF)
3. Train classification model
4. Predict job categories
5. Rank candidates using confidence scores

---

## 📊 Output
- Resume classification system
- Candidate ranking based on ML score
- Job role prediction results

---

## 🧠 Key Learning
- Natural Language Processing (NLP)
- Text classification models
- Resume ranking systems
- Real-world HR automation logic
